# 🐾 班级宠物屋

面向小学老师的班级积分管理工具，用“领养宠物 + 成长升级”的方式激励学生。

本仓库当前实现的是 **MVP 前端雏形**（无后端，使用 `localStorage` 持久化），包含：

- 多班级切换（示例数据）
- 学生卡片网格（搜索/排序占位）
- 领养宠物弹窗（示例宠物）
- 加分/扣分弹窗（示例规则）
- 设置与帮助弹窗（含“数据导入/导出/重置”）

## 技术栈

- 前端：Vue3 + Vite + TailwindCSS + Pinia
- 数据：localStorage（支持导入/导出 JSON）

## 本地运行

```bash
npm install
npm run dev
```

## 功能列表（规划）

- ✅ 班级管理（多班级切换）
- ✅ 学生管理（单个/批量添加）
- ✅ 50种宠物 × 10级成长阶段
- ✅ 积分加分/扣分（单个/批量）
- ✅ 撤回操作
- ✅ 小卖部（徽章兑换商品）
- ✅ 光荣榜排行
- ✅ 评价记录（支持撤回）
- ✅ 多主题切换
- ✅ 徽章墙
