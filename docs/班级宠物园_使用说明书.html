1. 产品简介

「班级宠物园」是一款面向课堂管理的轻量工具：通过 加分/扣分评价 推动学生的正向行为，同时将积分与“宠物成长/徽章/小商店兑换”等机制结合，提升参与度与持续性。

本软件默认无需登录，数据保存在本机浏览器（localStorage）。同时支持手动导入导出、记录导出、以及自动备份到本地文件（浏览器支持时）。

2. 快速开始（3 分钟上手）

  1. 打开网页后，先确认右上角班级名称（点击可进入 班级与学生管理）。
  2. 在 班级与学生管理 中：新增班级（可选）、添加学生或批量导入学生名单。
  3. 回到主界面：未领养的学生卡片显示 🥚，点击卡片或“领养宠物”为学生选择宠物。
  4. 给学生评价：点击已领养学生卡片 → 选择加分/扣分规则 → 完成一次评价。

3. 主界面说明

3.1 顶部功能区

  • 搜索学生：按姓名或学号筛选
  • 排行榜：查看徽章/等级等排序
  • 小商店：设置商品与兑换
  • 评价记录：查看最近 50 条，并支持导出
  • 撤回评价：撤回最近一次评价
  • 批量评价：对多名学生同时评价
  • 设置与帮助：规则管理、成长设置、数据管理等

3.2 学生卡片

  • 未领养：显示 🥚，点击领养
  • 已领养：显示宠物与等级、积分与徽章；点击卡片进入加分/扣分；左上角「⟳」支持更换宠物（更换后会清空积分）

4. 班级与学生管理

入口：点击顶部班级名或在设置中进入。

4.1 班级列表

  • 支持新增/编辑/删除班级；当前使用班级标记“当前使用”
  • 编辑/删除按钮为悬浮显示（hover）

4.2 学生列表

  • 支持新增学生（姓名必填，学号可选且需为纯数字）
  • 支持批量导入：每行一位（姓名 学号），学号可省略
  • 编辑/删除按钮为悬浮显示（hover）

5. 领养与更换宠物

  • 未领养：点击卡片或“领养宠物”
  • 已领养：点击卡片左上「⟳」更换
  • 更换宠物会清空该学生积分，以保证激励公平性

6. 加分/扣分评价（规则选择）

  1. 点击已领养的学生卡片（或批量模式）进入评价
  2. 选择分类（学习/行为/健康/其他）
  3. 点击规则卡片，即完成一次加分/扣分

7. 加减分规则管理

入口：设置 → 加减分规则。

  • 左侧：分类目录 + 规则统计（加分/减分条数）
  • 右侧：规则卡片（图标/名称/分值/范围标签），支持编辑/删除
  • 新增指标：图标、名称、分类、范围（全部/本班）、导向（奖励/扣分）、分值
  • 分值颜色：加分绿色、减分红色

8. 小商店（兑换）

  • 商品列表：展示可兑换商品（徽章作为货币）
  • 记录：查看兑换记录
  • 管理商品：新增/编辑/删除商品（图标、价格、库存、启用等）

9. 排行榜

默认按徽章数、等级、积分等排序（以当前版本界面为准）。

10. 评价记录（最近 50 条）

  • 查看最近 50 条：学生、规则、分类、时间、分值
  • 单条撤回（若该条记录支持）
  • 导出 Excel：当前导出为 CSV（Excel 可直接打开，且带 BOM 避免中文乱码）

11. 撤回评价

  • 顶部“撤回评价”：撤回最近一条
  • 评价记录“撤回”：撤回指定记录
  • 旧版本历史数据可能因缺少“操作前状态”而无法撤回

12. 数据管理

12.1 导入/导出（JSON）

  • 复制导出：复制 JSON 到剪贴板
  • 下载导出：下载 JSON 文件
  • 导入数据：粘贴 JSON 导入（会覆盖当前数据）

12.2 自动备份到本地文件（可选）

  • 需要浏览器支持 File System Access API（如 Chrome/Edge）
  • 首次选择备份文件并授权；支持开启/关闭、立即备份、清除目标、设置备份间隔
  • 清除目标仅清除浏览器记住的文件句柄，不会删除磁盘文件

13. 常见问题（FAQ）

Q1：数据会丢吗？

默认在浏览器 localStorage。建议定期“下载导出 JSON”，并在浏览器支持时开启“自动备份到本地文件”。

Q2：为什么目标文件不显示完整路径？

浏览器出于安全限制，通常无法读取完整路径，只能显示文件名或“已选择/未选择”。
